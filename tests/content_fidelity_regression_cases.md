# Content Fidelity Regression Cases v0.8.0

## Case 1: 运动零食长文

输入摘要：
用户提供一篇关于运动零食、66万人荟萃分析、WHO建议、31%、37%、可穿戴设备、每天2分钟、每天4分钟、每天3–4分钟、VILPA、HIIT类似通路的中文文章。

必须识别：

- 主题：运动零食 / 少量运动也有收益
- 主线：从不动到动收益最大
- 研究线 1：66万人荟萃分析
- 研究线 2：可穿戴设备 / 日常短促高强度活动
- 数字：31%、37%、24%、2分钟、4分钟、3–4分钟
- 机制：可能类似 HIIT，可能改善心肺功能、促进肌肉代谢、降低慢性炎症
- 行动建议：等水开深蹲、上楼快爬、赶路快走

必须避免：

- 不得把“几分钟”和“31%”强行合成封面承诺，除非 evidence ledger 显示同一 source anchor 同时支持它们。
- 不得把可穿戴设备研究和66万人荟萃分析写成同一个研究。
- 不得把“相关”写成确定因果。
- 不得把“每天3–4分钟”写成“每周几分钟”。
- 不得把“低于建议量”“达到建议量”“2–3倍建议量”混成同一组。
- 不得把机制推测写成确定机制。

推荐封面：

Title: “别等下次了”
Subtitle: “每天几分钟运动零食，也算动起来”
Points:

- “从不动到动，收益最大”
- “短促运动，也可能有收益”

推荐内容页：

Page 2:
Title: “先动起来，最划算”
Subtitle: “66万人研究：少量运动也有收益”
Points:

- “低于建议标准：死亡风险↓31%”
- “达到2–3倍标准：↓37%”
- “最大跃迁：从0到1”

注意：这里的 31% 按用户原文绑定到“低于建议标准”这条 claim，不得擅自改成外部事实，除非用户明确要求联网核查。

Page 3:

Subtitle: “可穿戴设备看见了日常运动片段”
Points:

- “每天2分钟：心血管死亡风险↓约24%”
- “每天4分钟：收益更明显”
- “3–4分钟：多项死亡风险下降相关”

Points:

- “上楼时：快一点爬楼”

## Case 2: 观察性研究因果误写

输入包含：

> “研究发现，A 与 B 风险下降相关。”

必须输出：

- “与风险更低相关”
- 或“相关下降”

必须避免：

- “A 会直接降低 B”
- “A 一定能预防 B”

## Case 3: 数字和时间窗口错配

输入包含：

- “每天3分钟”
- “每周150分钟”

两个不同建议。

必须避免：

- “每周3分钟达到150分钟效果”

## Case 4: 机制推测

输入包含：

> “研究人员认为，可能通过改善代谢发挥作用。”

必须输出：

- “可能改善代谢”

不得输出：

- “通过改善代谢发挥作用”作为确定事实。
