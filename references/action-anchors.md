# Action Anchors v0.8.10

The main subject must perform one action that explains the page content rather than serving as decoration.

Examples:

- putting a phone away for a sleep page;
- checking a food label for a nutrition page;
- walking after a meal for an activity page;
- organizing a desk for a life-order page;
- pointing to an independent text panel for an explanation page.

Choose a woman, man, student, older adult, family, animal, object-led scene or no character according to the source content. Keep each hand tied to at most one action target.
