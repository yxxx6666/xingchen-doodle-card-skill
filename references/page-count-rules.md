# Page Count Rules

## Default

Create a cover plus content pages.

## Length-based rules

### Very short input

If input is one sentence or one small idea:

- 1 cover
- 2 content pages

### Medium input

If input has 3–6 clear points:

- 1 cover
- 3–5 content pages

### Long input

If input is a long article or notes with many points:

- 1 cover
- 5–6 content pages
- compress aggressively

## Upper limit

Default maximum: 7 total pages.

Reason:

- Chinese text must be visible inside image_gen output.
- Too many pages increase inconsistency.
- Too much text reduces legibility.

## Page rule

Each page = one complete idea.

Do not split a single sentence fragment into a page.
Do not put multiple unrelated ideas on one page.
