# Image Gen Only Contract

## Core contract

This Skill is a single image generation closed-loop system.

The final visual card must be generated by image_gen only.
Illustration, scene, Chinese title, Chinese body text, optional English note, and composition must be produced in the same image generation pass.

## Absolute prohibitions

Do not use:

- PIL
- Canvas
- SVG
- HTML or CSS rendering
- browser screenshots
- local deterministic text layout
- local Chinese typography rendering
- separate illustration generation followed by text overlay
- post-processing text overlay
- programmatic drawing of text
- any non-image_gen visual generation

## Allowed output when image_gen is unavailable

If the current environment cannot call image_gen directly, output image_gen prompts only.
Do not replace image_gen with local rendering.

## Required prompt clause

Every page prompt must include:

```text
Chinese text integrated into image, generated by image_gen in the same pass. Do not omit the Chinese title or body text. No typography overlay, no post-processing, no external text rendering, no PIL, no Canvas, no SVG, no HTML.
```

## Acceptance criteria

A page is valid only if:

1. It is one complete image_gen prompt.
2. Chinese title is included verbatim in the prompt.
3. Chinese body text is included verbatim in the prompt.
4. Text placement is described inside the prompt.
5. No local rendering method is proposed.
6. No post-processing step is proposed.
